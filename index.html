ご挨拶

はじめまして。

カラフルでやさしい世界を届ける、小さなデザイン工房です。
「見る人の心が、ふっと明るくなるようなものをつくりたい」
そんな想いから、イラストからデザイン、コーディングまで、Webサイトをまるごと制作しています。

やわらかく、でも伝わる。
親しみやすく、でもちゃんと信頼できる。
そんな"ちょうどいい"を、一緒にカタチにしていけたら嬉しいです。

サービス内容

01

イラスト制作

見るだけで気持ちがほぐれるような、ポップでやさしいイラストを描いています。WebサイトやSNSはもちろん、グッズやアイコンなどにもご活用いただけます。

View MORE

02

Webデザイン

イラストを活かしながら、見る人に伝わるデザインを心がけています。レイアウト・配色・動きまで含めて、「世界観が伝わる」Webサイトを一緒に組み立てていきます。
あたたかく、でもスッキリ。見やすさとワクワクを両立したデザインを提案します。

03

Webサイト制作

HTML / CSS / JavaScriptを中心に、デザインをWeb上でしっかり再現します。
ノーコードツール（STUDIOなど）での実装も対応可能。ご希望に応じて、柔軟な形でサイトをお届けします。
